# 4Browser Keyboard Shortcuts Reference

## Tab & Window Management

| Shortcut | Action | Platform |
|----------|--------|----------|
| Ctrl+T | New tab | Windows/Linux |
| Cmd+T | New tab | macOS |
| Ctrl+N | New window | Windows/Linux |
| Cmd+N | New window | macOS |
| Ctrl+W | Close tab | Windows/Linux |
| Cmd+W | Close tab | macOS |
| Ctrl+Shift+W | Close window | Windows/Linux |
| Cmd+Shift+W | Close window | macOS |
| Ctrl+Shift+T | Reopen closed tab | Windows/Linux |
| Cmd+Shift+T | Reopen closed tab | macOS |
| Ctrl+Shift+N | New Incognito window | Windows/Linux |
| Cmd+Shift+N | New Incognito window | macOS |

## Navigation

| Shortcut | Action | Platform |
|----------|--------|----------|
| Alt+← | Back | Windows/Linux |
| Cmd+← | Back | macOS |
| Alt+→ | Forward | Windows/Linux |
| Cmd+→ | Forward | macOS |
| Ctrl+R | Reload | Windows/Linux |
| Cmd+R | Reload | macOS |
| Ctrl+Shift+R | Hard reload (clear cache) | Windows/Linux |
| Cmd+Shift+R | Hard reload | macOS |
| Ctrl+L | Focus address bar | Windows/Linux |
| Cmd+L | Focus address bar | macOS |

## Developer Tools

| Shortcut | Action | Platform |
|----------|--------|----------|
| F12 | Open DevTools | All |
| Ctrl+Shift+I | Open DevTools | Windows/Linux |
| Cmd+Option+I | Open DevTools | macOS |
| Ctrl+Shift+J | Open Console | Windows/Linux |
| Cmd+Option+J | Open Console | macOS |
| Ctrl+Shift+C | Inspect element | Windows/Linux |
| Cmd+Option+U | Inspect element | macOS |

## Tab Navigation

| Shortcut | Action |
|----------|--------|
| Ctrl+Tab | Go to next tab |
| Ctrl+Shift+Tab | Go to previous tab |
| Ctrl+1 through Ctrl+8 | Jump to tab 1-8 |
| Ctrl+9 | Jump to rightmost tab |
| Ctrl+Alt+Right | Jump to next open window |
| Ctrl+Alt+Left | Jump to previous open window |

## Zoom

| Shortcut | Action |
|----------|--------|
| Ctrl++ | Zoom in |
| Ctrl+- | Zoom out |
| Ctrl+0 | Reset zoom to default |

## Misc

| Shortcut | Action |
|----------|--------|
| Space | Scroll down one page |
| Shift+Space | Scroll up one page |
| Home | Go to top of page |
| End | Go to bottom of page |
| Alt+D | Jump to address bar |
| Alt+U | View page source |

---

**Note**: On macOS, `Ctrl` is typically replaced with `Cmd` (⌘) and `Alt` with `Option` (⌥)
